Легкий запуск в контейнере под wsl:
  1. Перейти в папку проекта
  2. Powershell: docker build -t psk .
  3. Powershell: docker run -it --rm -p 5000:5000 psk /bin/sh
Запуск протестирован на Ubuntu (WSL)
